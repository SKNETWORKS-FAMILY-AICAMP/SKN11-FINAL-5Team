# 목표
아래 **‘사업기획서 마크다운 포맷’**에 맞춰,
**제공된 데이터와 시장 통찰력**을 활용해
모든 항목을 **빈칸 없이** Markdown 형식으로 작성하라.

# 출력 포맷 (실제 Markdown 사용)
아래 포맷을 그대로 따라라.

# [아이템명] 사업 기획서

## 창업 아이디어 요약
아이템 핵심 아이디어를 간결하게 요약하라.

## 시장 조사 요약
시장 규모, 트렌드, 경쟁사, 고객 유형을 마크다운 표와 함께 정리하라.

## 비즈니스 모델 (Lean Canvas) : [아이템명] 사업 모델
아래 9개 항목을 **마크다운 표**로 작성하라.
모든 내용은 실제 사업 환경에 맞게 구체적으로 작성할 것.

| 항목                            | 내용 (구체적 사례, 수치, 설명 포함) |
| ----------------------------- | ---------------------- |
| 문제 (Problem)                  | 고객이 겪는 주요 문제 3가지 작성    |
| 아이디어 (Solution)               | 문제를 해결할 3가지 구체적 방안     |
| 가치 (Unique Value Proposition) | 한 문장으로 강력한 차별성 표현      |
| 핵심활동 (Key Metrics)            | 성공 측정 핵심 지표 2~3개      |
| 관계 (Unfair Advantage)         | 타사가 따라 하기 어려운 경쟁우위     |
| 고객 (Customer Segments)        | 타깃 고객층/페르소나 정의         |
| 비용 구조 (Cost Structure)        | 주요 비용 항목 기재            |
| 채널 (Channels)                 | 고객에게 알리고 전달하는 경로       |
| 수익원 (Revenue Streams)         | 수익이 발생하는 구조            |

**※ 표와 리스트는 반드시 마크다운 문법만 사용.**
